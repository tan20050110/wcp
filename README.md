# World Cup 2026 Predictor

A production-grade World Cup prediction platform with real-time match monitoring, comprehensive team analysis, and hybrid statistical/ML predictions.

## Quick Start

```bash
docker-compose up -d
# Backend: http://localhost:8000
# Frontend: http://localhost:5173
```

## Tech Stack

- Frontend: React 18 + TypeScript + Vite + TailwindCSS
- Backend: FastAPI + SQLAlchemy + PostgreSQL
- Prediction: Poisson + XGBoost + Monte Carlo
- Cache: Redis
